## **1주차 – 파이썬 기초 & 디버깅 연습**
**파일:** `코칭스터디_12기_1주차_딸쥬밀.ipynb`

- 기본 문법 오류 수정 (반복문, 조건문, 리스트 인덱싱 등)
- 모듈 import 오류 해결
- 한국 나이/만 나이 계산 프로그램 구현

## **2주차 – 조건문, 함수, 입력 처리 실전 문제**
**파일:** `코칭스터디_12기_2주차_딸쥬밀.ipynb`

### ✔ Q1. 성적 등급 출력 – `grader(name, score)`
- 점수에 따라 등급 출력
- `if / elif` 조건문 활용
- 이름과 등급을 함께 출력

### ✔ Q2. 월급 기반 연봉·세후 연봉 계산 – `yearly_payment(monthly_payment)`
- 월급 입력 → 연봉(세전/세후) 계산
- 세율 구간별 차등 적용

### ✔ Q3. 버스 요금 계산 – `bus_fare(age, payment_type)`
- 아동/청소년/성인/경로 구분
- 카드/현금 요금 차이 반영

### ✔ Q4. 가위바위보 게임 – `rcp(my)`
- 문자열 기반 가위바위보 입력
- 컴퓨터 랜덤 선택 생성
- 승·패·무승부 판단 및 예외 처리

## **3주차 – 함수, 반복문 응용**
**파일:** `코칭스터디_12기_3주차_딸쥬밀.ipynb`

### ✔ 구구단 출력 – `gugudan`
- 특정 단만 출력
- 홀수 곱셈만 출력
- 제한 조건 `i * 8 <= 50` 포함

### ✔ 범위 내 짝수 및 중앙값 판별 – `find_even_number`
- n~m 범위 리스트 생성
- 중앙값 계산 및 짝수 여부 판별

---

## **4주차 – 문자열 파싱 & 파일 입출력**
**파일:** `코칭스터디_12기_4주차_딸쥬밀.ipynb`

### ✔ 전화번호 유효성 검사 – `wrong_guest_book`
- 방명록 문자열 파싱
- `010-XXXX-XXXX` 형식 검증
- 잘못된 전화번호만 추출
- 결과 파일 `방명록.txt` 생성

---

## **5주차 – 콘솔 게임: 베스킨라빈스 31**
**파일:** `코칭스터디_12기_5주차_딸쥬밀.ipynb`

### ✔ 게임 함수 – `bs31`
- 플레이어가 1~3개의 숫자를 입력
- 입력 검증 (연속 증가 여부 등)
- 컴퓨터 차례 자동 생성
- 31을 말하면 패배하는 게임 로직 구현

---

## **6주차 – 자료 처리 & 간단한 데이터 분석**
**파일:** `코칭스터디_12기_6주차_딸쥬밀.ipynb`

### ✔ 고려/조선 왕 이름 교집합 – `king`
- 두 왕조 이름 문자열 파싱
- 공통 이름 출력 및 개수 계산

### ✔ 영업 사원 실적 분석 – `sales_management`
- 사원별 실적 NumPy 평균 계산
- 평균 실적 기준 내림차순 정렬
- 상위 2명 평균 5 이상 → 보너스 대상
- 평균 3 미만 → 면담 대상

### ✔ 주식 수익률 계산 – `stock_profit`
- 종목 문자열 파싱 (종목/수량/매수가)
- 판매가와 비교해 수익률 계산
- 수익률 내림차순 정렬
